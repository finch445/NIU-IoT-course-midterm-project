# NIU-IoT-course-midterm-project

## Introduction
1. Switching LED on/off by Sound Detector
2. LED Brightness adjust by Potentiometer
3. LED Brightness adjust by Photoresistance
4. Home Alert mode

## Tool
- Raspberry Pi 3B+ x1
- MCP3008 x1
- LED x5
- Buzzer x1
- Photoresistance x1
- Potentiometer x1
- Sound Detect Sensor x1
- Motion Detect Sensor x1
- Jumper wires
